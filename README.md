# Sales-Analysis-excel-dashboard
Built an interactive Excel dashboard to analyze retail sales performance, profitability, customer behavior, and delivery efficiency using real-world sales data.

# Tools Used
Microsoft Excel,
Pivot Tables,
Pivot Charts,
Slicers & Timeline,
XLOOKUP,
VLOOKUP,
INDEX-MATCH,
SUMIFS, COUNTIFS, AVERAGEIFS,
Conditional Formatting
# Key Analysis
Regional Sales Performance,
Product Performance Analysis,
Profitability Analysis,
Monthly Sales Trends,
Customer Segmentation,
Delivery Performance,
Salesperson Performance,
Return & Discount Analysis
# Dashboard Features
KPI Cards (Revenue, Profit, Orders, Avg Delivery Days),
Interactive Slicers,
Dynamic Charts,
Business Insights
# Key Insights
Identified top-performing regions and products,
Analyzed profit impact of discounts and promotions,
Evaluated delivery efficiency and return rates,
Measured salesperson and customer performance
